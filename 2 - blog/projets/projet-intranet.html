THALI est un centre de thalassothérapie situé au bord de l’Océan Atlantique. Il accueille des clients en cure toute l’année et dispose d’un hôtel pour l’hébergement des curistes qui le souhaitent.

L'objectif de ce projet était de créer une application de gestion d'excursions. Un excursion est est un trajet constitué de plusieurs étapes qui sont des passages dans des lieux-dits. Il fallait pouvoir afficher des fenêtres et faire fonctionner des champs de texte et des boutons pour gérer les informations d'excursions stockées dans une base de donnée.

Etape 4 : Visualisation des données d’une excursion

Dans cette partie du projet nous allons concevoir notre premier bouton interactif. Ce bouton aura pour but d'afficher les informations de l'excursion sélectionnée. On utilise des fenêtres JDialog pour simplifier l'aspect visuel puis on adapte le code faire fonctionner les éléments.

Etape 5 : Création d'une excursion

Maintenant que nous savons comment faire un bouton qui affiche du contenu, nous allons pouvoir faire un bouton qui pourra créer une nouvelle expédition. Nous avons désormais besoin de procéder à une requête SQL pour insérer les valeurs que l'utilisateur entre dans les champs de texte pour ajouter l'expédition à la base de donnée.
