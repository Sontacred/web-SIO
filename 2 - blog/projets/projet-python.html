Gestion de logs

L'objectif de ce projet était de lier l'utilisation de python et de SQL pour traiter des fichiers de logs
Le projet s'est déroulé en 9 étapes :

Etape 1 : Génération de pseudonymes à partir d'un fichier Etape 2 : Mise en place de la base de données en SQL Etape 3 : Analyse des fichiers textes de login Etape 4 : Écriture d'une requête SQL pour valider un jeu d’essai Etape 5 : Analyse des URL les plus consultées par utilisateur Etape 7 : Génération de scripts SQL pour l'insertion de données Etape 8 : Génération de scripts SQL pour l'insertion de données Etape 9 : Rédaction d’un mode opératoire
